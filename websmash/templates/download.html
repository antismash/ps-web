Download antiSMASH standalone

Stand-alone versions of antiSMASH 2.0 are available for Windows, Linux and Mac OSX.
  • Windows
  • Linux
  • Mac OSX
  • Extras

The current standalone release is antiSMASH 2.0.2 (May 8th, 2013)

Please select the installer for your operating system on the left. Under "extras", you can find some scripts that can aid in using antiSMASH

Windows XP/Vista/7 downloads

antiSMASH v2.0.2 installer package (32-bits)

antiSMASH v2.0.2 installer package (64-bits)

To be able to run antiSMASH on Windows, you will also need to install the following packages:

  • Microsoft Visual C++ 2008 Redistributable Package
  • Java
  • Python

The legacy antiSMASH v1.2.2 installer is also still available.

antiSMASH v2.0.2 installer script

Max OS X, for versions 10.6 (Snow Leopard) and higher

antiSMASH v2.0.2 installer package

To be able to run antiSMASH on Mac OS X, you will also need to install Cairo and libxml2.

These can be acquired by installing MacPorts:

  • Then, to get Cairo, type 'sudo port install cairo' in the terminal.
  • To get libxml2, type 'sudo port install libxml2' in the terminal.

The legacy antiSMASH v1.2.2 installer is also still available.
