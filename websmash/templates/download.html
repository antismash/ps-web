Download antiSMASH standalone

Stand-alone versions of antiSMASH 3.0 are available for Linux.
  • Linux
  • Source
  • Extras

The current standalone release is antiSMASH 3.0.4 (August 17th, 2015)

Please select the installer for your operating system on the left. Under "extras", you can find some scripts that can aid in using antiSMASH

Debian 8 "jessie"

antiSMASH v3.0.4 installer script

Docker container

There are two Docker containers available for antiSMASH 3.0.4: antismash/standalone (See instructions) and antismash/standalone-lite (Bring your own PFAM/ClusterBlast databases, see instructions).

Old versions

Legacy Ubuntu installers for antiSMASH v1.2.2 and antismash v2.1.1 are also still available, but not actively supported anymore.

Install from source

If none of the pre-made versions works for you, you can always install from source. Either download the tarball or git clone https://bitbucket.org/antismash/antismash.git.
Check out the instructions for a manual install.

Scripts that can aid in using antiSMASH:

pHMM seed alignments
Seed alignments used for building pHMMs used in secondary metabolite cluster finding
analysis plug-in
Template code for creating your own antismash 2.0 analysis plug-ins
output plug-in
Template code for creating your own antismash 2.0 output plug-ins
format_embl
Script for preparing an EMBL file from a TXT file with gene annotations and a DNA sequence in FASTA format
batch_antismash
Script for batch-processing multiple files in antiSMASH version 1.x
